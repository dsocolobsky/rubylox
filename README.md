# Rubylox

This is a Ruby 3.x interpreter of the Lox programming language from
[Crafting Interpreters](https://craftinginterpreters.com/)

## Features/Chapters Implemented
- [x] Scanning
- [x] Parsing
- [x] Evaluating
- [x] Statements
- [x] Expressions
- [x] Control Flow
- [x] Function Calling
- [x] Function Definition
- [x] Resolving and Binding
- [x] Classes
- [ ] Inheritance
